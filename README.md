# Básico de Javascript
Este repo tem como objetivo de ser um conjunto de princípios básico da linguagem de programação javascript.

### Assuntos abordados:
- `Variáveis`
- `Tipos de Dados`
- `Conversão de Dados`
- `Operadores`
- `Funções`
- `Condições`
- `Ciclos`
- `Spread Operator`
- `JSON`
- `Métodos de Strings`
- `Arrays`
- `Métodos de Arrays`
- `for of` e `for in`
- `Object Date()`
- `Desestrututação`
- `Arrow Functions`
- `Callbacks`
- `Assincronismo`
- `Closures`
